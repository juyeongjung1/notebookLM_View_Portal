# Googleスプレッドシート設定手順

ポータルサイトのコンテンツ（動画・音声リンク）を管理するためのGoogleスプレッドシートの設定手順です。

## 1. スプレッドシートの作成

Googleドライブで「新規」→「Googleスプレッドシート」を作成し、以下の**ヘッダー行**（1行目）を入力してください。

| A列 | B列 | C列 | D列 | E列 |
|------|------|------|------|------|
| title | category | type | url | description |

## 2. データの入力（2行目以降）
各コンテンツを1行ずつ入力します。

| title | category | type | url | description |
|-------|----------|------|-----|-------------|
| IT基礎-1章（解説動画） | IT基礎 | video | https://www.youtube.com/watch?v=xxxxx | 1章の動画です |
| IT基礎-1章（音声解説） | IT基礎 | audio | https://drive.google.com/file/d/xxxxx/view | 1章の音声です |
| IT基礎-2章（解説動画） | IT基礎 | video-drive | https://drive.google.com/file/d/xxxxx/view | Googleドライブの動画 |

### typeの種類
| type値 | 説明 | URLの例 |
|--------|------|---------|
| `video` | YouTube動画 | `https://www.youtube.com/watch?v=xxxxx` |
| `video-drive` | Googleドライブの動画 | `https://drive.google.com/file/d/xxxxx/view` |
| `audio` | Googleドライブの音声 | `https://drive.google.com/file/d/xxxxx/view` |

## 3. スプレッドシートを「ウェブに公開」

1. スプレッドシートの上部メニューから **「ファイル」→「共有」→「ウェブに公開」** を選択
2. 「リンク」タブで以下を設定：
   - 対象: **シート1**（または該当シート）
   - 形式: **カンマ区切り形式（.csv）**
3. **「公開」** ボタンをクリック
4. 表示された **URLをコピー** します

## 4. ポータルサイトにURLを設定

コピーしたURLを、`app.js` の以下の箇所に貼り付けます：

```javascript
const SHEET_CSV_URL = 'ここにコピーしたURLを貼り付け';
```

変更後、GitHubにPushすると反映されます。

## 5. 運用方法

- **コンテンツの追加**: スプレッドシートに新しい行を追加するだけ！
- **コンテンツの修正**: スプレッドシートの該当行を編集するだけ！
- **コンテンツの削除**: スプレッドシートの該当行を削除するだけ！
- **反映**: ポータルサイトをリロード（F5）するだけで最新データが表示されます

> **注意**: 「ウェブに公開」のデータはGoogleのキャッシュにより、更新から反映まで最大5分程度かかる場合があります。
