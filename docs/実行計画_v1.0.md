# 研修復習用ポータルサイト 実行計画
**バージョン**: 1.0
**日付**: 2026-03-04

## 1. 目的
研修受講者が、スマートフォンなどから「超簡単」に復習用の動画や音声解説（NotebookLMリンク）へアクセスできる専用ポータルサイトを構築する。

## 2. 実装方針とシステム構成
- **UI/UX**: ワンタップで再生画面へ遷移できるよう、スマートフォンに最適化されたモダンで美しいUI（Vanilla CSS・プレーンなHTML/JS）を採用。
- **データ管理**: NotebookLMへの自動アクセス・リンク取得はGoogleのセキュリティ（Bot対策）により実質不可のため、設定ファイル（`data.js`）へ手動でURLを貼り付ける方式とする。
- **公開方法**: GitHubやその他の静的ホスティングサービスを利用して無料かつ大量の同時接続（100名以上）に耐える構成とする。

## 3. 作成予定ファイル
- `index.html` : ユーザー向け画面（UIの骨組み）
- `style.css` : スタイル定義
- `app.js` : 画面の動的生成スクリプト
- `data.js` : 動画・音声のリンクデータを管理（管理者が編集するファイル）
